# Personal Portfolio & Systems Sandbox

A high-performance, interactive portfolio built to showcase frontend engineering, algorithm visualization, and system design concepts. 

## Tech Stack
* **Framework:** React 19 + TypeScript
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **Data Visualization:** Recharts

## Key Features
* **Algorithm Visualizer:** Implements and visualizes DFS, BFS, Greedy, and MRV heuristic algorithms solving Sudoku configurations in real-time.
* **Predictive Dashboard:** A financial insight UI featuring OLS (Ordinary Least Squares) regression forecasting for liquidity trajectories.
* **Theme Management:** Custom dark/light mode implementation using Context and session storage caching.

## Local Development
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
